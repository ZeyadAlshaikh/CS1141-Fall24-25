- Steps:
    - import
        - import javax.swing.*;
        - import java.awt.*;
    - extends JFrame
    - Declare your containers
    - Declare your components
    - in your constructor:
        1- set layout of your JFrame  [ setLayout(new FlowLayout());]
        2- set container with layout [ pnl.setLayout(new FlowLayout());]
        3- add components to your containers [pnl.add(btn);]
        4- add your panel to your JFrame [ add(pnl); ]
        5- repeat 2-4 for other panels
        6- set your title [ setTitle("Title");]
        7- set size for your JFrame [ setSize(300, 200);]
        7- set visible to true [ setVisible(True);]
        8- set default close operation [ setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); ]
    - Now you can use it as any other class
        - i.e String str = new String();
        - GUIDemo gui = new GUIDemo();
